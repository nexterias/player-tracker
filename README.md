# Player Tracker

**Assist in player tracking.**

> [!Warning]
> This plugin supports **Folia** only.

## Requirements

- [Folia](https://github.com/PaperMC/Folia)
    - API Version: **1.20**
- JDK 17

## Commands

| Command                | Description                                                    |
|------------------------|----------------------------------------------------------------|
| `/set-target <player>` | Start tracking by entering the target player name in <player>. |
| `/clear-target`        | Stop tracking.                                                 |

## Build

```shell
./gradlew build
```
